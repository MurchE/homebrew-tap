# homebrew-tap

Homebrew tap for Murch's Mac software.

## Install

```sh
brew tap MurchE/tap
```

### HeyClaude

Quick Claude answers from your terminal.

```sh
brew install murche/tap/heyclaude
```

### Naggler

Attention ledger — track where your focus actually goes.

```sh
brew install --cask murche/tap/naggler
```
